# AI Design Assistant Plugin

A prototype of an AI-assisted plugin for a design workflow, focused on generating and editing layout elements.

### Goal

The project explored how LLMs and automation could reduce repetitive design work: generating layout variants, editing existing elements, applying predefined design rules, and helping designers move faster inside an established workflow.

### What I Built

- Designed the concept and technical architecture of the AI-assisted design workflow.
- Built a working prototype for layout generation and editing operations.
- Implemented prompt-driven transformations for selected layout elements.
- Prepared the integration approach for connecting the plugin to a broader design production pipeline.
- Validated the idea through a functional prototype that could be extended into a production plugin.

### Stack

Python, OpenAI API, REST APIs, prompt engineering, plugin architecture, workflow automation.

### My Role

Lead AI Engineer / Prototype Developer. I designed the architecture, implemented the prototype, and validated the core AI-assisted editing flow.
