## Feature Pages Content

Контент для отдельных страниц, описывающих ключевые разделы настроек и функционала.

### 1. Neurons
**Tagline:** The Intelligence Engine.
**Description:**
Configure the brains behind the operation. "Neurons" are specialized AI agents assigned to specific roles—from the "Director" that manages the conversation to specialized experts for coding or vision.
*   **Universal Compatibility:** Use models from OpenAI, Anthropic, Google, Groq, or run fully local models via Ollama and LlamaCPP.
*   **Role-Based Assignment:** Assign cheaper, faster models for simple tasks and powerful reasoning models for complex logic.
*   **Custom Neurons:** Create your own custom agents with specific system prompts and parameters.

### 2. Memory
**Tagline:** Context is Everything.
**Description:**
hAI remembers so you don't have to. Our multi-layered memory architecture mimics human cognition to provide relevant context without overwhelming the AI.
*   **Short-Term & Long-Term:** Seamlessly transitions from immediate conversation context to long-term storage.
*   **Activity Watch:** Automatically logs and summarizes your computer activity (active windows, files), so the agent understands what you were working on yesterday or last week.
*   **Private & Local:** All memory is stored locally on your machine using SQLite. Your history stays with you.

### 3. Vision
**Tagline:** Seeing What You See.
**Description:**
Give hAI eyes. With Vision capabilities, the agent can perceive your screen, understand UI elements, and analyze images.
*   **Snapshots:** The agent captures the state of your screen to understand the context of your request.
*   **UI Analysis:** Detects buttons, fields, and text on the screen to interact with applications that don't have APIs.
*   **Computer Vision Models:** Utilizes specialized vision models (like Claude 3.5 Sonnet or GPT-4o) for high-precision visual tasks and UI navigation.

### 4. Hearing
**Tagline:** Always Listening, Always Ready.
**Description:**
Advanced audio processing allows hAI to hear and understand you clearly, even in noisy environments.
*   **Speech-to-Text (STT):** High-accuracy transcription using cloud providers (Deepgram, OpenAI) or local Whisper models for privacy.
*   **Wake Word:** Custom wake word detection (e.g., "Hey Computer") for instant, hands-free activation.
*   **VAD:** Intelligent Voice Activity Detection ensures the agent listens only when you are speaking.

### 5. Speaking
**Tagline:** A Natural Voice.
**Description:**
hAI responds back naturally. Choose a voice that fits your preference and enjoy fluid conversation.
*   **Text-to-Speech (TTS):** Industry-leading synthesis from ElevenLabs, Cartesia, or fast local options like Piper and Silero.
*   **Real-Time Response:** Optimized for low latency to create a feeling of genuine conversation.
*   **Customization:** Adjust speed, pitch, and select from a wide library of voices to give your agent a personality.

### 6. Action
**Tagline:** From Thought to Execution.
**Description:**
This is where hAI truly becomes a manager. It is equipped with a suite of tools to perform tasks on your computer and the web.
*   **Core Tools:** Web search, file management, clipboard access, and terminal execution.
*   **UI Scenarios:** Complex automation scripts that control the mouse and keyboard to perform multi-step tasks in any application.
*   **Python Executor:** The agent can write and execute Python code to solve mathematical problems, process data, or create charts on the fly.
*   **MCP Ecosystem:** Extend functionality infinitely by adding community or custom MCP servers.
