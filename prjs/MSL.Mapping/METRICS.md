# Метрики проекта: МСЛ Картирование

Проект выполнялся в рамках магистерской дипломной работы и являлся, по сути, R&D-прототипом для внутреннего использования во ВНИСИ.

**Сбор продуктовых или командных метрик не велся**, так как это был исследовательский проект с одним основным разработчиком.

Основным критерием успеха была **функциональная полнота**:
- Корректность работы алгоритма интерполяции данных.
- Способность системы обрабатывать и визуализировать данные с реального заезда МСЛ без сбоев.
- Соответствие визуализации на карте (треки, тепловая карта) ожиданиям инженеров-светотехников.
